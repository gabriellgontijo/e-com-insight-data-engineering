 E-Com Insight

Sobre o Projeto
O E-Com Insight é um projeto de engenharia de dados que propõe a construção de uma arquitetura moderna para um e-commerce, com foco em análise de dados batch e streaming.

  Objetivo
Planejar uma arquitetura escalável, desacoplada e eficiente, capaz de suportar ingestão, processamento e análise de dados para tomada de decisão e sistemas de recomendação.

 Arquitetura
A solução adota o modelo Lakehouse com abordagem Lambda, permitindo o processamento de dados em batch e em tempo real.

 Tecnologias Utilizadas
- Apache Kafka
- Airbyte
- Apache Spark
- Apache Airflow
- Delta Lake
- MinIO / S3
- Power BI
- Docker
